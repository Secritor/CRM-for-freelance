# Мини CRM для фрилансеров

        Проект представляет CRM для фрилансеров

## Stack

        1. React, TS
        2. Redux
        3. React-router
        5. recharts / chart.js
        6. Vitest + React Testing Library (unit-тесты)

## Include

    [x] 1. Окно для регистрация и аутентификация

    [x] 2. Страница клиентов. На карточке имя, компания, почта, номер телефона, адрес, дополнительная информация.
        возможность добавить или удалить пользователя, изменить существующую информацию. Есть возможность поиска клиента через поисковую строку.

    [x] 3. Страница сделок. В верхней части страницы введена информация с выручкой за текущие проекты, за завершенные проекты,
        и общий счетчик. Имеется таблица с такими полями как:
            3.1 Краткое описание сделки
            3.2 Имя клиента и\или название организации
            3.3 Заработная плата
            3.4 Статус сделки
            3.5 Время выполнения
            3.6 Доступные действия - Изменить, Добавить, Удалить сделку.

    [ ] 4. Страница с календарем в котором можно запланировать событие

    [ ] 5. Страница с аналитикой включающая в себя:
            5.1 Возможность просмотреть аналитику за n-период 1/6/12 месяцев
            5.2 Информация об общем доходе за период
            5.3 Информация о среднем доходе за месяц
            5.4 Общее количество сделок за период
            5.5 Количество текущих клиентов
            5.5 Линейный график с трендом дохода за месяцы
            5.6 Круговая диаграмма с информацией о процентном соотнешии статусов заявок
            5.7 Гистограмма прибыли от клиентов
            5.8 Детализированная таблица дохода от клиентов включающая в себя:
                5.8.1 Имя клиента
                5.8.2 Доход
                5.8.3 Количество сделок
                5.8.4 Средний доход
                5.8.5 Дата последний сделки

## Testing

    Unit-тесты: `npm run test` (watch) или `npm run test:run` (single run).
    Подробнее — см. [TESTING.md](./TESTING.md).

## Features

        1. Сбор продвинутой аналитики сделок:
            1.1 Процент роста дохода относительно предудущего периода
            1.2 Процент конверсии
            1.3 Средний период сделки
            1.4 Процент удержания клиентов
        2. Возможность создания целей внутри приложения и отображение текущей статистики относительно целей
